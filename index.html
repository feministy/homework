To get started, create an account on JSBin. I recommend connecting it to your Github account so it's easy for you to access later. You'll be writing all of your JavaScript code in JSBin and saving it to review later.

Tests are required for your code to be reviewed. JavaScript tests can be written in Qunit. We will cover other methods for writing JavaScript tests later.

Before you begin...

You need to be familiar with:

  • HTML and HTML5 semantic elements
  • CSS and CSS3 basics
  • JavaScript syntax
  • JavaScript data structures
  • jQuery

I don't expect you to know everything, but you should be comfortable enough with those things that you know where to go to look stuff up. If you're not, complete the JavaScript (the whole thing), HTML, and CSS tracks on Codecademy. I would also suggest that you read [Eloquent JavaScript](http://eloquentjavascript.net/contents.html), but we will be covering a bit of that during the weekly lessons.
