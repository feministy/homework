To get started, fork the repository. Clone it to your local machine: 

$ git clone https://github.com/YOUR_USERNAME/homework.git

Follow the instructions for each exercise and submit your code for review by submitting a pull request. Save your code inside of the submissions/your_name folder.

If you are writing JavaScript or Ruby, tests are required for your code to be reviewed. JavaScript tests can be written in Qunit and Ruby tests should be written in RSpec.
